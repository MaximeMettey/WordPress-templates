== Description ==

Restaurant Premium est un thème WordPress professionnel et moderne spécialement conçu pour les restaurants, cafés, bistros et tous types d'établissements de restauration. Il offre une expérience utilisateur exceptionnelle avec un design élégant et des fonctionnalités spécialement adaptées à l'industrie de la restauration.

= Caractéristiques principales =

* Design moderne et professionnel avec une palette de couleurs élégantes (or et gris foncé)
* Section hero personnalisable avec image de fond
* Types de contenu personnalisés pour les éléments de menu
* Types de contenu personnalisés pour les témoignages
* Catégories de menu (entrées, plats principaux, desserts, etc.)
* Système de prix pour les éléments de menu
* Responsive et mobile-friendly
* Support des images à la une
* Personnalisation facile via le Customizer WordPress
* Zones de widgets multiples (sidebar et 3 zones de footer)
* Menus de navigation personnalisables
* Support des couleurs personnalisées
* Optimisé pour le SEO
* Compatible avec les plugins majeurs de WordPress
* Traduction ready
* Support HTML5
* Code propre et bien documenté

= Zones de widgets =

* Sidebar principale
* Footer Area 1
* Footer Area 2
* Footer Area 3

= Menus =

* Menu principal (navigation)
* Menu footer

= Options de personnalisation =

Via le Customizer WordPress, vous pouvez personnaliser :
* Logo du site
* Section Hero (titre, sous-titre, bouton, image de fond)
* Couleurs principales (couleur primaire, couleur secondaire)
* Informations de contact (téléphone, email, adresse)

== Installation ==

1. Dans votre administration WordPress, allez dans "Apparence" > "Thèmes"
2. Cliquez sur "Ajouter"
3. Téléchargez le fichier ZIP du thème
4. Cliquez sur "Installer maintenant"
5. Une fois installé, cliquez sur "Activer"

= Configuration manuelle =

1. Téléchargez le fichier ZIP du thème
2. Décompressez-le
3. Uploadez le dossier 'restaurant-premium' dans '/wp-content/themes/' via FTP
4. Dans votre administration WordPress, allez dans "Apparence" > "Thèmes"
5. Activez le thème Restaurant Premium

== Foire aux questions ==

= Comment personnaliser les couleurs du thème ? =

Allez dans "Apparence" > "Personnaliser" > "Theme Colors" pour modifier les couleurs principales du thème.

= Comment ajouter des éléments au menu du restaurant ? =

Dans votre administration WordPress, vous trouverez une nouvelle section "Menu Items" où vous pouvez ajouter vos plats avec leur description, prix et image.

= Le thème est-il responsive ? =

Oui, Restaurant Premium est entièrement responsive et s'adapte parfaitement à tous les types d'écrans (desktop, tablette, mobile).

= Puis-je utiliser ce thème avec un constructeur de pages ? =

Oui, le thème est compatible avec les principaux constructeurs de pages WordPress comme Elementor, Gutenberg, etc.

= Le thème est-il compatible WooCommerce ? =

Le thème peut être utilisé avec WooCommerce pour la vente en ligne, bien que des ajustements de style puissent être nécessaires.
